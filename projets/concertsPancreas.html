Concerts du Pancréas

Présentation du projet

Afin de noter nos compétences en développement web, un projet en deux parties a été mis en place pour le cinquième semestre de mon BUT. Le but de ce projet est de développer une API Symfony ainsi qu'une interface Vue.js afin de gérer un site de gestion d'événements de notre choix. J'ai donc réalisé ce travail avec trois camarades de classe et nous avons choisi de faire un gestionnaire de concerts.

Ma participation

J'ai donc eu l'occasion de travailler sur l'API en créant la base données ou en gérent certaines requêtes avec par exemple l'utilisation de StateProcessor. Côté interface, je me suis principalement concentré sur l'initiation de certains aspects et sur la page utilisateur regroupant des informations de l'API ainsi que des données récupérées d'un autre projet du semestre 5.

Les compétences utilisées

Programmation en PHP - Utilisation de Vue.js - Utilisation de Symfony - Programmation d'API - Utilisation de GitLab - Travail en équipe - Gestion d'une base de données en SQL

Accéder au site

Vous pouvez accéder au site Concerts du Pancréas pour explorer les fonctionnalités disponibles. Le projet n'étant pas une SAE mais un simple projet noté, nous avions moins de temps afin de le réaliser.

Il est également possible d'accéder au site Annuaire, le projet lié au profil de l'utilisateur.
